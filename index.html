7. Explain the differences between statements “heap is complete” and “heap is full”
Heap is complete: It is complete if each level of the tree is filled except the last one and if
each level is filled from left to right
Heap is full: It is full if it is a tree in which every node other than the leaves has two
children.

Common Ancestor/Parent) of the 2 elements.
17. public void lca (Tree root, int n1, int n2) {
while (root!=NULL){
if(root.data() > n1 && root.data() > n2)
root = root.left();
else if(root.data() < n1 && root.data() < n2)
root = root.right();
else break;
}
System.out.println(root.data());
}
